Compliance & Security

HIPAA by design. Not by afterthought.

Encryption

AES-256 at rest, TLS 1.3 in transit. All PHI encrypted end-to-end with no cleartext exposure at any processing layer.

Authentication

MFA required for all practitioner and admin accounts. Argon2id password hashing. Rate-limited auth endpoints. 15-minute session timeout.

Audit Trail

Complete audit logging with 6-year retention. Every access, assessment, and report generation is tracked with user identity and timestamp.

BAA Coverage

Business Associate Agreements with all infrastructure vendors. Render, database, and API providers all under signed BAAs.

Data Isolation

Per-organization data partitioning. CSAA forensic data separated from CI clinical data. No cross-contamination between pathways or tenants.

Access Control

Role-based access with four tiers: practitioner, supervisor, admin, and classified (CSAA). IP allowlisting available for admin portals.

Deployment

Flexible deployment for any infrastructure.

NOUS deploys as a cloud-hosted SaaS platform or as an on-premise installation for organizations with strict data residency requirements. Both configurations deliver identical clinical capability.

The platform is currently live on Render.com (Starter tier) with an AWS migration path documented for enterprise-grade infrastructure including HIPAA-eligible compute, dedicated VPC, and custom domain configuration.

  • Cloud SaaS — fastest deployment, managed infrastructure
  • AWS enterprise — HIPAA-eligible, dedicated VPC, custom domain
  • On-premise option — for data sovereignty requirements
  • FHIR API integration with Epic, Cerner, AllScripts
  • SSO via SAML 2.0 / OAuth for organizational identity
  • Custom LOGOS cultural calibration for patient populations
  • Dedicated CNAP training cohorts for clinical staff
