NOUS COGNITIVE INTELLIGENCE PLATFORM

Platform Glossary

What each component does and what the terminology means

Beverly Index LLC — Reference Document  |  March 2026

The NOUS ecosystem spans a scientific discipline, a theoretical framework, a classification system, a multi-layer processing platform, and four assessment pathways. This glossary defines each term at the functional level — what it does, who uses it, and how it relates to the rest of the system.

For detailed explanation of NOSS sector scoring and how to interpret analysis results, see the NOSS Sector Lexicon.

Scientific Foundation

THE SCIENCE BEHIND THE PLATFORM
Computational Psychodynamics
The Scientific Discipline
Foundational Science
The scientific field that studies how cognitive states encode in communicative output — how the mind reveals itself through language and voice. Computational Psychodynamics is to the NOUS platform what genomics is to a DNA sequencing platform. It is the parent discipline from which every downstream technology derives its scientific validity.
NOST
Neural Oscillation Signature Theory
Foundational Theory
The theoretical framework that explains how the brain's electrical rhythms (neural oscillations) manifest in language and vocal output. NOST maps specific cognitive states to specific patterns in communicative behavior. Every algorithm in the NOUS ecosystem operates according to NOST's theoretical predictions. Currently under peer review at Frontiers in Psychology (Manuscript ID 1834364) and published on SSRN (DOI: 10.2139/ssrn.6447802).
NOSS
Neural Oscillatory Spectroscopy System
Classification System
The 12-sector classification system derived from NOST theory. NOSS categorizes cognitive patterns across twelve diagnostic domains, each scored continuously on a 0–100% activation scale. Every assessment produces activation percentages across all 12 sectors simultaneously — a complete cognitive signature.
S1 Mood S2 Anxiety S3 Psychotic S4 Neurocognitive S5 Developmental S6 Optimal S7 TBI/CTE S8 Neurodegenerative S9 Captivity S10 Substance S11 Dissociative S12 Personality
Lense-Thirring Precession
Psychological Integration Model
Theoretical Extension
NOST's model for how the mind processes and integrates significant psychological experiences over time. Named after the Lense-Thirring effect in physics (frame-dragging around a massive rotating body, demonstrated in a landmark 2024 MIT/Nature study), this framework tracks how unprocessed psychological material "orbits" and gradually integrates with the core self — observable through oscillatory patterns in linguistic output. The model classifies integration trajectories into six patterns that predict therapeutic progress.

The Platform

PRODUCT & ASSESSMENT PATHWAYS
NOUS
The Cognitive Intelligence Platform
The Product
The operational platform that houses every technology in this glossary. NOUS takes the science (Computational Psychodynamics), applies the theory (NOST), uses the classification system (NOSS), and delivers it through four pathways as a deployable clinical and forensic product. The name comes from the Greek νοῦς, meaning "mind" or "intellect."
CI
Cognitive Intelligence Pathway
Pathway 1 — Clinical
The clinical mental health analysis pathway. A practitioner assesses a subject's cognitive state through narrative input and receives a comprehensive evaluation across all 12 NOSS sectors with Beverly Index scoring, treatment intelligence, and longitudinal tracking.
CSAA
Cognitive Signature Authenticity Assessment
Pathway 2 — Forensic
The forensic authenticity assessment pathway. CSAA analyzes any narrative to determine whether it reflects genuine memory recall or constructed fabrication, producing a quantified Authenticity Index (0–100). Designed for intelligence, defense, legal, and insurance applications.
VOCA
Vocal Oscillation Cognitive Archive
Pathway 3 — Monitoring
The continuous cognitive monitoring pathway. VOCA tracks each individual's cognitive signature over time using periodic language samples, detecting meaningful deviations from that person's own baseline — not population averages. When cognitive markers shift in ways suggesting clinical change, VOCA detects it before self-report or clinical observation.
RTCA
Real-Time Call Analysis
Pathway 4 — Live Audio
Live analysis of telehealth sessions, crisis calls, and ambient speech — detecting cognitive signatures in real-time without structured intake. The distribution infrastructure for NOUS at scale.

Processing Layers

THE SEVEN-LAYER TECHNOLOGY STACK
AAE
Adaptive Assessment Engine
L1 — Input & Orchestration
The intelligent front door. AAE accepts any narrative input (dreams, memories, conflicts, daily events), determines what clinical domains are activated, decides whether more information is needed, and generates adaptive follow-up prompts until clinical confidence is achieved. It is the only component the subject interacts with directly.
LOGOS
Cross-Cultural Calibration System
L2 — Calibration
The system that knows who is speaking before analyzing what they're saying. LOGOS identifies the speaker's linguistic origin, cultural communication norms, and prosodic baseline — ensuring that culturally expressive speech patterns aren't misread as clinical pathology, and that restrained communication norms aren't misread as emotional suppression.
REMIEL Engine
Multi-Channel Analysis System
L3 — Extraction
The core analysis engine. REMIEL receives calibrated narratives and analyzes language and vocal patterns across 8 simultaneous channels, extracting the cognitive signature embedded in the words.
Ch1 Dreams Ch2 Conflict Ch3 Emotional Ch4 Future Ch5 Self Ch6 Speech Ch7 Creative Ch8 POA (Voice)
POA
Prosodic Oscillation Analysis
REMIEL Channel 8
REMIEL's dedicated voice analysis channel. POA extracts acoustic-prosodic features — pitch, rhythm, energy, vocal quality, pause architecture, and articulation — enabling Text-Voice Divergence Detection: identifying when what someone says contradicts how they say it. POA is the foundation for VOCA continuous monitoring.
R2X Bridge
Translation Engine
L4 — Translation
The translation layer that sits between extraction and output. R2X receives REMIEL's analysis and determines what cognitive states the extracted features represent, generating the NOSS sector activation scores, visual parameters, and audio parameters that feed every downstream output.

Output & Visualization

SCORING, VISUALIZATION & IMMERSIVE EXPERIENCES
Beverly Index
Composite Cognitive Health Metric
L6 — Scoring
The single number at the top of every analysis. The Beverly Index integrates all twelve NOSS sectors, coherence, integration dynamics, and diagnostic probability into one score on a 0–100 scale. It is a composite cognitive health metric — not a diagnosis, but an overall reading of how the language maps to known patterns of cognitive function and dysfunction.
Clinical Action Priority (0–100) Diagnostic Probability Coherence (0.0–1.0) Integration Dynamics Integration Pattern Confidence (0–100) Authenticity Index (CSAA only)
DreamWeaver
2D Neural Visualization
L5a — Visual Output
Turns cognitive data into visual representations. Pattern density, color mapping, and structural complexity all correspond to the subject's cognitive signature. In forensic mode, DreamWeaver creates the NOST Signature Map — the visual cognitive fingerprint used in CSAA reports.
Memoria Sonata
Neural Audio Synthesis
L5b — Audio Output
Turns cognitive data into sound. Healthy cognitive patterns sound harmonious and warm; distressed patterns sound tense, dissonant, or fragmented. Memoria Sonata also sonifies integration progress over time, providing an auditory timeline of therapeutic change.
Empyrean
3D Cognitive Topology
L7 — Immersive Visualization
The full three-dimensional universe of a subject's cognitive signature. NOSS sectors are rendered as distinct 3D regions orbiting a central core that represents the subject's processing style. Empyrean supports temporal playback — animating changes over weeks or months to show healing, deterioration, or stability. The name comes from the medieval concept of the highest heaven — the clearest possible view.
Treatment Intelligence
NOSS-Matched Clinical Navigation
Output Layer
The clinical navigation system that matches a subject's NOSS activation profile to appropriate treatment modalities, providers, and clinical trials — both local (geographic match) and worldwide (best-in-class). Treatment Intelligence is informational, never prescriptive. Clinical decisions remain with the licensed practitioner.

Key Concepts

OPERATIONAL TERMINOLOGY
BLUF Dashboard
Bottom Line Up Front
Dashboard Architecture
The first clinical element a practitioner sees. A military-derived communication principle: give the decision-maker the most important information first, then let them drill down. The BLUF displays a severity-coded signal (CRITICAL / HIGH / MODERATE / LOW / CLEAR) and a one-sentence clinical verdict. Every subsequent layer is progressive disclosure — the practitioner "pulls" deeper data only when they want it.
TRAIT Baseline
Individual Reference Profile
Longitudinal Concept
A subject's established personal cognitive baseline, built from their own prior sessions. All subsequent assessments are measured against the individual's TRAIT profile — not population norms. This is what makes NOUS longitudinally powerful: every person is measured against themselves.
Delta Mode
Change Detection Protocol
Assessment Protocol
An assessment mode activated for returning subjects with an established TRAIT baseline. Delta Mode prioritizes change detection over absolute thresholds — measuring how much the subject's current cognitive signature has shifted from their personal baseline. Essential for TBI monitoring, treatment response tracking, and neurodegenerative progression.
Integration Dynamics
Processing Trajectory Classification
Beverly Index Component
The system that classifies how a subject's mind is processing psychological material. Based on the Psychological Lense-Thirring Precession model, Integration Dynamics produces one of six pattern classifications that predict therapeutic trajectory:
NORM-I — Healthy integration STUCK — Stalled processing DISORG — Disorganized processing HYPER — Hyperactive cycling AVOID — Avoidant trajectory DISS-T — Dissociative trajectory
Authenticity Index
CSAA Narrative Authenticity Score
CSAA Pathway Only
The CSAA pathway's primary output score (0–100). Measures the degree to which a narrative reflects authentic recall versus fabricated construction. High scores (70+) indicate authentic recall. Low scores (below 40) indicate fabrication signatures. The score is reproducible — the same narrative produces the same score on reassessment.
CNAP
Certified NOST Analysis Practitioner
Certification Program
The certification program for practitioners who complete training in NOUS platform operation and Computational Psychodynamics methodology. CNAP certification validates competency in administering NOUS assessments, interpreting Beverly Index scores, reading NOSS sector activations, and integrating platform output with clinical judgment.
